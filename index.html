HTML

  • The head element contains information about the webpage including metadata which describes the information in the HTML document.
  • The body element represents the visible content shown to the user including paragraph and list elements as well as any other visible elements.
  • br and section elements help to organize seen information.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.
  • A shadow will do a shadow effect given an X and Y value, shadowing then to the right and down respectively
  • Colors can be described by name or Hex#
  • CSS can be inline (within HTML element), internal (within HTML head element and applies to all elements within a page), or external (seperate code that is linked in along with other metadata)

Git

  • git status: checks what branch we are currently on
  • git checkout: change branch
  • git checkout -b (branch name): creates new branch with whatever name you give it
  • cd: change directory (change file)

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
